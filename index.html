정보 접근성 정책을 도출하기 위한
업종 별 웹 접근성 실태 파악

웹 접근성은 장애인, 노인 등과 같은 사용자도 웹사이트에서 제공되는 콘텐츠를 용이하게 이용할 수 있도록 보장하는 것을 의미한다.
비장애인은 웹사이트에서 제공되는 텍스트와 이미지, 영상 등을 접했을 때 어려움 없이 인식할 수 있지만, 시각, 청각 등 기능에 장애가 있는 사람에게는 그렇지 못하다.
현재 웹 사이트 사용 빈도가 높은 사업체 사이트들이 기본적인 웹 접근성을 지켜 제작되었는지 파악하고, 향후 웹 정보 접근성 관련 정책을 도출하기 위해 기획하였다.

  • ❓ 모든 웹사이트의 웹 접근성이 전체적으로 잘 지켜지고 있는가?
  • ❓ 업종 별 웹 접근성이 부족한 항목은 무엇인가?
  • ❓ 전체적으로 잘 지켜지고 있지 않은 항목이 있는가? 있다면 무엇인가?
  • ❓ 웹 접근성이 제일 좋은 업종, 제일 좋지 않은 업종은 어디인가?
  • ❓ 업종 특성이 웹 접근성 결과에 미치는 영향이 있는가?
  • ❓ 웹 접근성의 4가지 원칙으로 봤을 때는 어떤가?
